## Dependencies

* Eigen3
* PCL
* OpenCV
* YAML-CPP

## Build This Program

```
git clone https://github.com/bzdfzfer/CloudLine2Plane.git
mkdir -p ~/CloudLine2Plane/build/ && cd ~/CloudLine2Plane/build/
cmake .. 
make -j8
```
